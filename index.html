Inventory-manager

Java-based basic inventory manager using MongoDB for persistent data storage.

Download .zip Download .tar.gz View on GitHub

Welcome the Inventory Manager Project Site

Inventory Manager is a simple inventory management system designed to demonstrate how to use the MongoDB driver for Java. The client is (obviously) written in Java and interfaces with MongoDB as a persistent data store.

This project is still under development.
